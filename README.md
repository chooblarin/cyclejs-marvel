## How to run

1. Create developer account on [Marvel](https://marvel.com/register)
2. Get an API key
3. Put `config.js` in `app/` directory like follows

```
export const config = {
  API_KEY: 'Your API Key'
}
```

4. `yarn install` and `yarn start`
